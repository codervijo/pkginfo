Beaglebone Black Board (BBB)
    Processor : AM335x 1GHz ARM Cortex-A8 from TI
    Ubuntu On BBB : https://beagleboard.org/p/robertcnelson-myopenid-com/ubuntu-6c7f26

    Getting Started : http://beagleboard.org/getting-started


 URL to download prebuilt image : https://beagleboard.org/latest-images
 Or from direct URL : https://debian.beagleboard.org/images/bone-debian-9.5-iot-armhf-2018-10-07-4gb.img.xz

 You can write that image on to the micro SD card using etcher.

 Before you start, you will need 
      1, micro-HDMI cable. Or micro-hdmi to HDMI adapter
      2, Micro SD card with image written on it
      3, 5V adapter for power
 

CPU Documentation
      1, AM335x Sitara Processors : http://www.ti.com/lit/ds/symlink/am3352.pdf
